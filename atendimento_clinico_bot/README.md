## Atendimento Clinico bot

### Objetivo

- agendar consultas clínicas
- prestar um serviço de anamnese simples entre especialistas
- ter um historico medico do paciente
- laudar um paciente
- prontuário simplificado
- gerar formulário de consulta
- gerar formulário de atendimento
